# Oakville Market Intel — Karolis Rushkys

Weekly luxury real estate market intelligence for Karolis's 5 GTA farm pockets.

**Live app:** see GitHub Pages URL

## Pages
- **This Week** — five-segment overview with key signals
- **Segment Detail** — full KPI strip, 4 charts, AI-written analysis
- **Compare** — side-by-side segment comparison
- **Trends** — 12-month arc charts
- **Newsletter Helper** — copy-paste blocks for newsletter draft
- **History** — prior period archive
- **Data Input** — Sunday data entry (TRREB Stratus paste/CSV)
- **Settings** — thresholds and configuration

## Sunday Workflow
1. Pull TRREB Stratus export for each segment
2. Open Data Input page → paste data → Submit
3. Open Newsletter Helper → copy paragraphs into newsletter draft
4. Edit to voice, send — done in 15 minutes

## Data
All data stored in browser localStorage on each device.
Seed data (Oakville March 2026, TRREB MLS) embedded in `shared.js`.

## Compliance
TRREB Stratus data — internal use only. Not for distribution.
CASL s.6(5) · RECO Rule 7 · §3 hard constraints apply.
